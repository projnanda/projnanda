## Foundational Architecture Papers

<div style="display: grid; gap: 20px; margin: 30px 0;">

  <div style="background: #ffffff; border: 2px solid var(--nanda-yellow); border-left: 6px solid var(--nanda-yellow); border-radius: 12px; padding: 25px 30px; box-shadow: 0 4px 15px rgba(255, 215, 0, 0.15); transition: all 0.3s ease;" onmouseover="this.style.boxShadow='0 8px 25px rgba(255, 215, 0, 0.3)'; this.style.transform='translateX(5px)';" onmouseout="this.style.boxShadow='0 4px 15px rgba(255, 215, 0, 0.15)'; this.style.transform='translateX(0)';">
    <h3 style="color: var(--nanda-black); margin: 0 0 12px 0; font-size: 1.15rem; border: none; padding: 0;">
      <a href="https://arxiv.org/abs/2507.14263" style="color: var(--nanda-black); text-decoration: none;">Beyond DNS: Unlocking the Internet of AI Agents</a>
    </h3>
    <p style="color: #555; line-height: 1.7; margin: 0 0 15px 0; font-size: 0.95rem;">
      <em>The foundational paper.</em> Details the design of the NANDA Index, AgentFacts schema, and Verified Agent Discovery. Introduces mechanisms for multi-endpoint routing, privacy-preserving queries, and rapid revocation in decentralized agent ecosystems.
    </p>
    <a href="https://arxiv.org/abs/2507.14263" style="display: inline-block; background: linear-gradient(135deg, #FFD700, #FFC700); color: #000; padding: 8px 20px; border-radius: 8px; text-decoration: none; font-weight: 600; font-size: 0.9rem; transition: all 0.3s ease; box-shadow: 0 2px 8px rgba(255, 215, 0, 0.3);">arXiv:2507.14263 &rarr;</a>
  </div>

  <div style="background: #ffffff; border: 2px solid var(--nanda-yellow); border-left: 6px solid var(--nanda-yellow); border-radius: 12px; padding: 25px 30px; box-shadow: 0 4px 15px rgba(255, 215, 0, 0.15); transition: all 0.3s ease;" onmouseover="this.style.boxShadow='0 8px 25px rgba(255, 215, 0, 0.3)'; this.style.transform='translateX(5px)';" onmouseout="this.style.boxShadow='0 4px 15px rgba(255, 215, 0, 0.15)'; this.style.transform='translateX(0)';">
    <h3 style="color: var(--nanda-black); margin: 0 0 12px 0; font-size: 1.15rem; border: none; padding: 0;">
      <a href="https://arxiv.org/abs/2506.12003" style="color: var(--nanda-black); text-decoration: none;">Upgrade or Switch: The Need for New Registry</a>
    </h3>
    <p style="color: #555; line-height: 1.7; margin: 0 0 15px 0; font-size: 0.95rem;">
      Analyzes why DNS fails for AI agents and weighs incremental upgrades against purpose-built registries. Uses analogies like dial-up to broadband to outline the technical deltas introduced by the agent paradigm.
    </p>
    <div style="display: flex; gap: 12px; flex-wrap: wrap;">
      <a href="https://arxiv.org/abs/2506.12003" style="display: inline-block; background: linear-gradient(135deg, #FFD700, #FFC700); color: #000; padding: 8px 20px; border-radius: 8px; text-decoration: none; font-weight: 600; font-size: 0.9rem; transition: all 0.3s ease; box-shadow: 0 2px 8px rgba(255, 215, 0, 0.3);">arXiv:2506.12003 &rarr;</a>
      <a href="../upgrade_or_switch.md" style="display: inline-block; background: #000; color: #FFD700; padding: 8px 20px; border-radius: 8px; text-decoration: none; font-weight: 600; font-size: 0.9rem; transition: all 0.3s ease;">Read Full Paper &rarr;</a>
    </div>
  </div>

  <div style="background: #ffffff; border: 2px solid var(--nanda-yellow); border-left: 6px solid var(--nanda-yellow); border-radius: 12px; padding: 25px 30px; box-shadow: 0 4px 15px rgba(255, 215, 0, 0.15); transition: all 0.3s ease;" onmouseover="this.style.boxShadow='0 8px 25px rgba(255, 215, 0, 0.3)'; this.style.transform='translateX(5px)';" onmouseout="this.style.boxShadow='0 4px 15px rgba(255, 215, 0, 0.15)'; this.style.transform='translateX(0)';">

    </p>
    <a href="https://arxiv.org/abs/2508.03113" style="display: inline-block; background: linear-gradient(135deg, #FFD700, #FFC700); color: #000; padding: 8px 20px; border-radius: 8px; text-decoration: none; font-weight: 600; font-size: 0.9rem; transition: all 0.3s ease; box-shadow: 0 2px 8px rgba(255, 215, 0, 0.3);">arXiv:2508.03113 &rarr;</a>

  <div style="background: #ffffff; border: 2px solid var(--nanda-yellow); border-left: 6px solid var(--nanda-yellow); border-radius: 12px; padding: 25px 30px; box-shadow: 0 4px 15px rgba(255, 215, 0, 0.15); transition: all 0.3s ease;" onmouseover="this.style.boxShadow='0 8px 25px rgba(255, 215, 0, 0.3)'; this.style.transform='translateX(5px)';" onmouseout="this.style.boxShadow='0 4px 15px rgba(255, 215, 0, 0.15)'; this.style.transform='translateX(0)';">

  <div style="background: #ffffff; border: 2px solid var(--nanda-yellow); border-left: 6px solid var(--nanda-yellow); border-radius: 12px; padding: 25px 30px; box-shadow: 0 4px 15px rgba(255, 215, 0, 0.15); transition: all 0.3s ease;" onmouseover="this.style.boxShadow='0 8px 25px rgba(255, 215, 0, 0.3)'; this.style.transform='translateX(5px)';" onmouseout="this.style.boxShadow='0 4px 15px rgba(255, 215, 0, 0.15)'; this.style.transform='translateX(0)';">
    <h3 style="color: var(--nanda-black); margin: 0 0 12px 0; font-size: 1.15rem; border: none; padding: 0;">
      <a href="https://arxiv.org/abs/2508.03095" style="color: var(--nanda-black); text-decoration: none;">Survey of AI Agent Registry Solutions</a>
